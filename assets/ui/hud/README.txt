HUD Window Textures
===================

Положите PNG-файлы с нужными именами в эту папку —
движок подхватит их автоматически. Если файл не найден,
окно отрисуется процедурно (ShapeRenderer).

STATUS WINDOW (верхний левый угол, 500x500 px)
  Файл:  ui/hud/status_window.png
  Рекомендуемый размер:  500 x 500 пикселей
  Назначение:  фон окна с HP-баром, уроном и жизнями

MINIMAP WINDOW (верхний правый угол, 500x500 px)
  Файл:  ui/hud/minimap_window.png
  Рекомендуемый размер:  500 x 500 пикселей
  Назначение:  фон окна с миникартой, названием тира и этажом

KEYS WINDOW (нижний правый угол, 250x250 px)
  Файл:  ui/hud/keys_window.png
  Рекомендуемый размер:  250 x 250 пикселей
  Назначение:  фон маленького окна с ключами

KEY ICON (внутри KEYS WINDOW)
  Файл:  ui/hud/key_icon.png
  Рекомендуемый размер:  64 x 64 пикселей (или кратно)
  Назначение:  иконка ключа — заменяет процедурный рисунок

Текстуры растягиваются точно под размер окна.
Рекомендуется RGBA8888, фильтр Nearest или Linear — по вкусу.
